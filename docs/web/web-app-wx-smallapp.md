## 目录结构
### page 文件夹
> 小程序页面, 快捷创建方式: `app.json` -> `pages` -> `直接添加路径保存`，就能直接生成`整个页面文件`了

相关文件
+ index.js  (某页面js逻辑)
+ index.json  (某页面数据)
+ index.wxml  (某页面html)
+ index.wxss  (某页面css)

### util 文件夹
> 存放一些工具方法

### 文件
+ app.js  (全局js逻辑)
+ app.json  (全局配置)
+ app.wxss  (全局css)
+ project.config.json (一些配置项目配置信息)
+ sitemap.json (设置爬虫文件)
